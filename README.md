# Bike Ride Usage Analysis (Power BI)

## Project Overview
This project analyzes bike ride usage data to understand customer behavior,
ride patterns, and peak usage times using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Key Insights
- Total rides exceed 5.5 million
- Members take more rides than casual users
- Electric bikes are used more frequently than classic bikes
- Casual riders have longer average ride durations
- Ride demand peaks during specific hours of the day

## Visualizations
- Total Rides KPI Card
- Rides by Membership Type
- Rides by Bike Type
- Average Ride Duration by Membership
- Rides by Start Hour

## Skills Demonstrated
- Data cleaning and transformation
- Date-time extraction
- Aggregations and measures
- Dashboard design and storytelling
